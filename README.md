# مشروع التفريغ الصوتي المحلي (Parakeet TDT STT)

هذا المشروع يوفر بيئة متكاملة وسريعة لتشغيل نموذج **Parakeet TDT 0.6B v3** (بصيغة ONNX) محلياً. 

[المصدر](https://huggingface.co/istupakov/parakeet-tdt-0.6b-v3-onnx)

[اقرأ هذا الملف](parakeet-stt/docs/parakeet.md)

تم تصميم المشروع ليعمل بكفاءة عالية على المعالج (CPU)، مع استخراج نصوص وترجمات دقيقة التوقيت متوافقة مع معايير الترجمة المرئية. كما يتفوق على نموذج وايسبر.

* يدعم النموذج 25 لغة أوروبية، وهي: البلغارية، الكرواتية، التشيكية، الدنماركية، الهولندية، الإنجليزية، الإستونية، الفنلندية، الفرنسية، الألمانية، اليونانية، المجرية، الإيطالية، اللاتفية، الليتوانية، المالطية، البولندية، البرتغالية، الرومانية، السلوفاكية، السلوفينية، الإسبانية، السويدية، الروسية، والأوكرانية.

---

## المميزات الأساسية
- **تحميل مباشر وذكي:** سكربت التحميل يجلب الملفات المباشرة من Hugging Face بدون الحاجة لمكتبات خارجية أو تسجيل دخول، مع دعم **شريط التقدم** و**استئناف التحميل** في حال انقطاع الاتصال.
- **تفريغ دقيق (TXT & SRT & VTT):** يقوم باستخراج النص الخام وملف ترجمة `SRT` و `VTT` ذو توقيتات مبنية بذكاء بناءً على علامات الترقيم والوقفات الطبيعية للمتحدث.

---

## طريقة التشغيل

### 1. المتطلبات
```bash
pip install "onnx-asr[cpu]" 
```

إذا كان لديك بطاقة رسوميات من NVIDIA وأردت تسريع التفريغ بشكل كبير، يمكنك تغيير الاعتماديات لدعم كرت الشاشة بتغيير `cpu` إلى `gpu`.

### 2. تحميل النموذج
قم بتشغيل سكربت التحميل لجلب أوزان النموذج محلياً (حوالي 3 جيجابايت). السكربت يدعم الاستئناف، فإذا توقف التحميل يمكنك إعادة تشغيله بآمان:
```bash
python download.py
```
*(سيتم حفظ النموذج في مجلد `local_model`)*

### 3. تشغيل التفريغ الصوتي
1. ضع الملف الصوتي `wav` الخاص بك في المجلد الرئيسي (وتأكد من الإسم داخل ملف `run.py`).
2. نفذ أمر التشغيل:
```bash
python run.py
```
